# Симулятор газового месторождения

Учебный проект: разработка математической модели куста газовых скважин. 

## Описание
Программный комплекс выполняет расчет рабочей точки системы «пласт - скважина - шлейф - ДКС» и моделирует динамику добычи газа. 

Структура проекта:
* src/ — пакет с модулями физики флюида (GERG-91), материального баланса, гидравлики (Дарси-Вейсбах) и решателем (scipy.optimize).
* test_file.py — скрипт для модульного тестирования.
* курсовая.ipynb — итоговый отчет: построение IPR/VLP кривых, моделирование на 180 дней и калибровка модели под промысловые данные.
* adapt_gdi_11-2025.csv — исторические данные для калибровки.

Стек: Python, NumPy, Pandas, SciPy, Matplotlib.
